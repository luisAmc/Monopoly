OBSERVACIONES:
	- El programa esta hecho en c++ y haciendo uso de la libreria SFML 2.1.
	- Fue creado sobre Ubuntu 13.04 64bits.


Antes de iniciar el programa, es necesario instalar la libreria SFML 2.1,
estas se pueden descargar de la pagina oficial: http://www.sfml-dev.org/download/sfml/2.1/, 
aparte, es necesario intalar unas bibliotecas de las que depende la libreria:

	sudo apt-get install libpthread-stubs0-dev
	sudo apt-get install libgl1-mesa-dev
	sudo apt-get install libx11-dev
	sudo apt-get install libxrandr-dev
	sudo apt-get install libfreetype6-dev
	sudo apt-get install libglew1.5-dev
	sudo apt-get install libjpeg8-dev
	sudo apt-get install libsndfile1-dev
	sudo apt-get install libopenal-dev
	sudo apt-get install cmake
	sudo apt-get install g++
